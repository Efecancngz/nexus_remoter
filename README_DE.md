# 🇩🇪 Nexus Remote Kommandozentrale

![Nexus Remote](https://images.unsplash.com/photo-1550751827-4bd374c3f58b?q=80&w=2070&auto=format&fit=crop)

**Nexus Remote** verwandelt Ihr Smartphone oder Tablet in eine leistungsstarke, KI-gestützte PC-Kommandozentrale. Unterstützt durch Google Gemini AI, übersetzt es natürlichsprachliche Befehle (wie „Öffne Spotify und spiele Rock“) in echte Aktionen auf Ihrem PC.

## ✨ Hauptmerkmale
*   **🧠 AI-Makros:** Versteht komplexe Befehle wie „Licht aus und Filmmodus starten“.
*   **🔒 Sichere Kopplung:** Verhindert unbefugten Zugriff durch ein dynamisches PIN-System.
*   **⚡ Geringe Latenz:** Sofortige Reaktionszeit über das lokale Netzwerk.
*   **🎨 Anpassbar:** Erstellen Sie Ihre eigenen Tasten, Farben und Symbole.

## 🚀 Einrichtung (Schritt für Schritt)
1.  **PC-Agent starten:** Führen Sie `NexusAgent.exe` auf Ihrem Computer aus.
2.  **Mobil verbinden:** Öffnen Sie die Weboberfläche.
3.  **Koppeln:** Klicken Sie oben links auf „NEXUS“. Geben Sie den PIN-Code und die IP-Adresse Ihres PCs ein.
4.  **Fertig!**
